Selectors

  1. * - Universal selector (One style for all web page elements)
  2. h3 - Type selectors (Selects all elements that have the given node name.)
  3. .red - Class selectors (Selects all elements that have the given class attribute.)
  4. #green - ID selectors (Selects an element based on the value of its id attribute.)
  5. [target="_blank"] - Attribute selector (Selects all elements that have the given attribute.)
  6. .strong, #em - Selector list (The "," selector is a grouping method that selects all the matching nodes.)
  7. a span - Descendant combinator (The " " (space) combinator selects nodes that are descendants of the first element.)
  8. a > div -
    Child combinator (The > combinator selects nodes that are direct children of the first element.)
  9. Selector b ~ em - this is General sibling combinator (The ~ combinator selects siblings. This means that the second element follows the first (though not necessarily immediately), and both share the same parent.)
  10. .a__neighbor + li - Adjacent sibling combinator (The + combinator matches the second element only if it immediately follows the first element.)
